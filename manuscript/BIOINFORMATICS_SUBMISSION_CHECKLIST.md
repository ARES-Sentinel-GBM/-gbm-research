# Bioinformatics Journal Submission Checklist

## ✅ Pre-Submission Requirements

### 1. Manuscript Requirements
- [x] Main manuscript (LaTeX or Word)
- [x] Abstract (≤250 words, structured)
- [x] Figures (5, high-resolution PDF)
- [x] References (Vancouver style)
- [x] Word count: ~4000-5000 words (Main text)

### 2. Supplementary Materials
- [ ] Supplementary Methods
- [ ] Supplementary Tables
- [ ] Supplementary Figures
- [ ] README for supplementary data

### 3. Required Documents
- [x] Cover Letter
- [ ] Author Contribution Form
- [ ] Conflict of Interest Disclosure
- [ ] Data Availability Statement
- [ ] Code Availability Statement

### 4. Repository Requirements
- [x] Code deposited in public repository (GitHub)
- [x] DOI assigned (via Zenodo or similar)
- [x] Repository link in manuscript

## 💰 Article Processing Charge (APC)

**Bioinformatics Open Access:**
- Standard APC: £3150 / €3650 / $4875
- **Waivers available** for corresponding authors from low-income countries
- **No page charges** for standard length manuscripts

**Subscription Model (no APC):**
- Free to publish
- 6-month embargo before OA
- Color figures: free online, may charge for print

## ✨ Submission Strategy

1. **Submit to bioRxiv first** ✅ DONE (MS ID: BIORXIV/2026/715908)
2. **Wait for bioRxiv approval** (24-72 hours)
3. **Submit to Bioinformatics** with bioRxiv DOI
4. **Mention bioRxiv version** in cover letter
5. **Respond to reviews** within 30 days

**Note:** Bioinformatics allows preprint submission (bioRxiv is not considered prior publication)
